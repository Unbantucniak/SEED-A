# VS Code 自学习自演化智能助手插件
## 项目原型系统

### 功能介绍
这是本项目的IDE原型系统，是一款VS Code插件，实现了经验自动积累、搜索、复用的完整功能，帮助开发者在编码过程中自动积累和复用历史开发经验。

### 核心功能
1. **经验自动积累**：自动保存成功的开发交互为结构化经验
2. **经验搜索**：支持自然语言搜索相关历史开发经验
3. **经验管理**：支持手动保存经验、查看经验详情、管理经验库
4. **智能推荐**：编码过程中自动推荐相关经验（待实现）

### 快速开始
#### 1. 启动后端服务
```bash
cd backend
pip install fastapi uvicorn
python app.py
```
后端服务默认运行在 `http://localhost:8000`

#### 2. 编译安装插件
```bash
# 安装依赖
npm install

# 编译代码
npm run compile

# 按F5启动插件开发调试窗口，或打包为vsix安装包
```

#### 3. 插件配置
打开VS Code设置，搜索"自学习自演化智能助手"，配置：
- `apiEndpoint`：后端服务地址，默认`http://localhost:8000`
- `autoSaveExperience`：是否自动保存成功交互为经验，默认开启

### 使用说明
#### 搜索经验
1. 按下`Ctrl+Shift+P`打开命令面板
2. 输入"智能助手：搜索相关开发经验"
3. 输入关键词搜索，点击结果查看详情

#### 手动保存经验
1. 选中要保存的代码片段
2. 打开命令面板，输入"智能助手：保存当前交互为经验"
3. 输入需求描述，保存成功

#### 查看经验库
打开左侧活动栏的"智能开发助手"面板，查看所有经验
